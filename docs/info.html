ROMY Ring‑Laser Monitor

Site Guide & Documentation

Purpose

This site provides near real‑time and recent historical monitoring products for the multicomponent ring‑laser gyroscope system ROMY (ROtational Motions in seismologY) near Munich, Germany. The focus is expert operational awareness: instrument health, data quality, dominant disturbances, and environmental coupling.

Using the Figures

  • Images auto‑refresh (5–60 min cadence depending on page). A small cache‑buster query string forces reload.
  • Click any figure (cursor turns pointer) to open a larger lightbox view.
  • If a figure fails to generate, a placeholder image ("oops") is shown; persistent placeholders indicate upstream processing issues.

Operational Notes

  • Grey / empty intervals in spectra denote missing or unusable data (not low signal).
  • Vertical red (MLTI) or yellow (maintenance) bands annotate intervention periods; interpret adjacent frequency excursions cautiously.
  • Backscatter correction line (black dashed) should converge toward cleaned signal; divergence flags calibration drift.
  • Environmental correlation: inspect tilt / temperature shifts when long‑term Sagnac drifts accelerate.

Data Access

For raw rotational / translational time series, contact the ROMY operations team or consult internal seedlink endpoints (not publicly listed here). Future public APIs may expose low‑latency rotational channels.
